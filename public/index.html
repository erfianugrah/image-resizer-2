Image Resizer Service

This is a Cloudflare Workers-based image resizing service. To use this service, append image parameters to your image URLs.

This endpoint serves images, not HTML. You need to specify an image path to get a response.

Basic Usage Examples:

Resize an image:

/path/to/image.jpg?width=400&height=300

Convert format:

/path/to/image.jpg?format=webp

Resize and adjust quality:

/path/to/image.jpg?width=800&quality=85

Using Akamai compatibility:

/path/to/image.jpg?im.resize=width:400,height:300&im.quality=85
